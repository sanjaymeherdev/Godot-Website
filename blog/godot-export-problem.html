üö´ Common Errors & Fixes

Error: "Failed to export project"

Fix: Go to Project ‚Üí Tools ‚Üí Create Android Build Template first, then try export again.

Error: "Keystore not found"

Fix: Re-browse to your .keystore file in Export settings. Make sure path is correct.

Error: "Invalid password"

Fix: Check Keystore Password and Keystore/Release Password match exactly.

Error: "APK installs but crashes"

Fix: Add missing permissions (INTERNET, ACCESS_NETWORK_STATE). Check if ads are causing crash.

Error: "App not installed" on phone

Fix: Uninstall debug version first, then install release version. They conflict.

Error: "No activities found"

Fix: Make sure you have Android module enabled in Export ‚Üí Android ‚Üí Gradle Build.
