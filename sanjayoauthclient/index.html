🎯 What is Automation Hub?

⚡ n8n Automation Platform

Automation Hub is a workflow automation platform built on n8n that connects your entire Google Workspace. Create custom automation flows that:

  • ✅ Process emails automatically
  • ✅ Sync spreadsheet data with other apps
  • ✅ Manage files and documents
  • ✅ Schedule meetings and send invites
  • ✅ Run custom Apps Script functions

No code required – Build workflows visually using n8n's drag-and-drop interface.

🔑 Requested Permissions – Why Each is Needed

📌 Every scope is required for specific n8n automation features

📧 Gmail Access

Scope: gmail.modify

Why we need it:

  • ✅ Trigger workflows from incoming emails (support tickets, orders, notifications)
  • ✅ Send automated replies – confirmations, follow-ups, reports
  • ✅ Organize inbox – auto-label, archive, or delete based on rules
  • ✅ Extract data from email content for other workflows

🔹 We only access emails you explicitly select for automation. No bulk reading or storage.

📊 Sheets Access

Scope: spreadsheets

Why we need it:

  • ✅ Read/write data – sync CRM, inventory, or form responses
  • ✅ Trigger workflows from spreadsheet changes (new row, updated cell)
  • ✅ Generate reports – populate sheets with automated results
  • ✅ Bulk data processing – import/export between systems

🔹 We only access sheets you select; no background scanning of all your spreadsheets.

📁 Drive Access

Scope: drive

Why we need it:

  • ✅ Auto-backup files – archive attachments, upload documents
  • ✅ Organize files – create folders, move/copy files based on workflow
  • ✅ Export/convert – generate PDFs, DOCX, or other formats
  • ✅ Share files – automate file sharing with team members

🔹 We access only files/folders you explicitly include in your automation rules.

📅 Calendar & Meet Access

Scope: calendar + calendar.events

Why we need it:

  • ✅ Schedule meetings – auto-create events with Meet links
  • ✅ Check availability – avoid double-booking
  • ✅ Send invites – add attendees automatically
  • ✅ Update events – reschedule or cancel based on workflow triggers

🔹 Meet links are auto-generated via Calendar API – no separate Meet scope exists.

⚙️ Apps Script Access

Scope: script.external_request, script.scriptapp, script.send_mail

Why we need it:

  • ✅ Extend automation – run custom scripts beyond n8n's native capabilities
  • ✅ Complex transformations – manipulate data with JavaScript
  • ✅ Cross-service actions – combine Gmail, Sheets, Drive in one script
  • ✅ Send emails – using Apps Script's built-in MailApp

🔹 Scripts are user-created and user-triggered; we never execute hidden scripts.

👤 User Identity (Basic Auth)

Scope: userinfo.email, userinfo.profile, openid

Why we need it:

  • ✅ Authenticate users – identify who is logged in
  • ✅ Personalize workflows – assign automation to specific users
  • ✅ Audit logs – track which user triggered each workflow
  • ✅ Display profile – show name/avatar in the dashboard

🔹 This is the "basic auth" replacement – OIDC instead of username/password.

⚙️ How Automation Works

1️⃣ Connect Your Google Account

You sign in with Google and grant the permissions listed above. These permissions enable Automation Hub to interact with your Workspace services.

2️⃣ Build Workflows (n8n)

Using n8n's visual editor, you create automation flows by connecting triggers (e.g., "new email") to actions (e.g., "update spreadsheet").

3️⃣ Workflows Run Automatically

Workflows execute on schedule, on trigger events, or on-demand. Each automation runs with the permissions you granted.

4️⃣ Data is Ephemeral

We process data in-memory during workflow execution. We do not store emails, files, or calendar data. Only metadata (like workflow logs) is saved.

🔒 Data Privacy & Handling

📌 What We Process

  • ✅ Emails – only those you select for automation triggers
  • ✅ Sheets – only spreadsheets connected to workflows
  • ✅ Drive files – only files/folders in your automation rules
  • ✅ Calendar events – only events created/modified by workflows
  • ✅ User profile – used for authentication and dashboard display

🚫 What We Never Do

  • ❌ Store emails, files, or calendar data beyond workflow execution
  • ❌ Share your data with third parties
  • ❌ Read all your emails/files – only explicit automation targets
  • ❌ Sell or monetize your data
  • ❌ Access data outside of your active automation workflows
