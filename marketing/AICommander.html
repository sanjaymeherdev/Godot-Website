The system

4 pieces. One system.

Each part does one job. Together they give you every AI tool, on every device, for free.

🎼 The Orchestrator

A Chrome extension that lives on your main PC. It opens DeepSeek, Claude, ChatGPT, Gemini, and Qwen in silent background tabs and controls them like a robot — sending your prompts, creating new chats, navigating chat history, and waiting for the full response before returning it. Uses your own logged-in accounts. No API keys needed.

🔁 The Relay Server

A private server that runs on your PC and bridges every remote client to the Orchestrator. It exposes a clean REST API — /prompt/:provider, /run, /status — protected by your own API key. Nothing passes through any third-party server. Your data never leaves your machines.

API key protected 100% private WebSocket + REST
3

💻 The Remote Extension

Install this Chrome extension on any second PC, connect it to your relay with the URL and API key, and you instantly have access to all 5 AI tools from that machine too. It has a full side panel for regular use — and a killer feature most people miss:

🤖 AI Quick Prompt ✕

Select any text on any webpage, hit a keyboard shortcut, and a quick prompt overlay pops up right there — pre-filled with your selected text. Ask AI about it without leaving the page. Ctrl+Enter to send.

Works on any webpage Keyboard shortcut trigger Select text → instant AI
4

📱 PocketAI — The Mobile App

A floating bubble that lives on top of every Android app. Connects to your relay and uses your PC's AI sessions. When your PC is off or unreachable, it automatically switches to Groq (Llama 3.3 70B) — so you always get a response. Think of PocketAI as the remote control for AICommander in your pocket.

Overlay on every app Groq auto-fallback Android · One-tap AI
