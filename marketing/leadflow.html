📜 Google Apps Script Setup

1 Create New Script
Go to script.google.com and click New Project.
2 Paste the Code Below
Delete any existing code and paste this: 
function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const { to, subject, html, fromName } = data;
    
    GmailApp.sendEmail(to, subject, '', { 
      htmlBody: html,
      name: fromName || 'LeadFlow'
    });
    
    return ContentService.createTextOutput(JSON.stringify({ 
      success: true, 
      message: 'Email sent' 
    })).setMimeType(ContentService.MimeType.JSON);
    
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({ 
      success: false, 
      message: error.toString() 
    })).setMimeType(ContentService.MimeType.JSON);
  }
}
3 Deploy as Web App
Click Deploy → New deployment
4 Configure Settings
⚙️ Web app
Execute as: Me (your email)
Who has access: Anyone
5 Deploy & Authorize
Click Deploy, then:
⚠️ CRITICAL - DON'T SKIP!
When the authorization popup appears:
• Click "Review Permissions"
• Click "Allow"
❌ If you skip authorization, emails will NOT send!
After authorizing, copy the generated URL.
6 Paste URL & Save
https://script.google.com/macros/s/.../exec
Paste it in the Apps Script URL field above and click Save All Settings.
7 ✅ Test It!
Click the 🧪 Send Test Email (Apps Script) button.
Check your inbox (and spam folder) for the test email.
