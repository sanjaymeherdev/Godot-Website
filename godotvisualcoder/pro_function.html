📘 User Guide ▶

🗂️ Script structure — always in this order

class_name (optional) → gives your script a reusable name across the project.
extends (required) → tells Godot what kind of node this script belongs to. Without it, nothing works.
Declarations → constants, exported vars, onready refs, signals — all before functions.
Functions → _ready() runs once on load · _physics_process(delta) runs every physics frame.

🏷️ Annotation quick reference

@export → variable appears in the Godot Inspector
@onready → gets a node reference when scene loads (use $NodeName)
const → value that never changes (use UPPER_CASE)
signal → custom event other nodes can listen to

🔀 Nested logic tips

After adding if / for / while, the next action auto-indents inside it. Use ◀ ▶ to adjust manually. Block openers show a darker background; else/elif show dashed borders and snap back to the correct indent automatically.

🔗 Condition builder (if / elif / while)

Each condition row has: not toggle · left operand · comparator · right operand. Click + AND/OR condition to chain multiple bools. Click the AND/OR badge between rows to toggle the operator.
